Dynamic Range Compression

These examples illustrate use of the DynamicsCompressorNode Two code examples are provided.
  1. Attack and release: shows how to visualize the effect of varying the ballistics, that is, the compressor's attack and release parameters.
  2. Compressor: shows a compressor in action. It displays a meter of the gain reduction while a user changes the parameters of a compressor that is applied to a source.
Code examples correspond to examples from Chapter 13 in the book 'Working with the Web Audio API'. Feel free to contact me at joshua.reiss@qmul.ac.uk
